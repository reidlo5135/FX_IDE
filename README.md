## [서비스 주요 기능]
- <b>Java Compile</b><br>
1. 사용자가 입력한 코드를 .java 파일로 변환
2. Java ProcessBuilder 를 이용하여 명령프로세스(cmd)를 통해 변환된 .java 파일 실행
3. 실행 결과를 Stream 으로 추출 후 컬렉션(ArrayList)에 추가 후 JavaFX Client에 출력
